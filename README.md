# Project: Recipes

This is a simple website built using only **HTML** as part of **The Odin Project** Foundation curriculum.

## About

This project is just a practice for fundamental of HTML by creating a multi-page recipe website.

The Website includes;
- Homepage with Links to recipes basically index
- Images of the food
- headings
- Paragraphs
- Lists both the structured and unstructured
- Links between pages

No CSS or JavaScript has been used in this project for now. The foucs is entirely on learnign HTML structure and semant
